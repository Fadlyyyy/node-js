perbandingan antara nodejs dan php :
php dibuat pada tahun 1994
nodejs 2009

php menggunakan engine zendengine
nodejs menggunakan engine google's v8 engine

untuk eksekusi kodenya php menggunakan multi-thread dan synchronous
kodenya disebut blocking, apabila mengerjakan satu hal akan berjalan sampai selesai dahulu.
baru mengerjakan hal proses berikutnya
namun dia adalah multi thread , yang melayaninya lebih dari 1

sedangkan nodejs yang melayaninya hanya 1, namun ia tidak perlu menunggu hingga selesai
1 proses ia bisa langsung ke proses berikutnya

php menggunakan web server apache atau / iis 
sedangkan nodejs berjalan diruntime sendiri.
(hal ini yang membuat proses deployment di nodejs tidak semudah diphp)

php mempunyai package manager yang bernama composer
nodejs mempunyai package manager yang bernama NPM
